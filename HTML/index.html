AcosoZeroBatiz
AcosoZeroBatiz es una empresa comprometida con la sana convivenvia, tiene como objetivo hacer que el acoso deje de ser un tema tan normalizado por todos. Esta empresa está desarrollando una página web para poder registrar estos casos de acoso; los principales programadores son: CECYT 9 Estos son: David Quirino, Cristian, Cristofer, Fernanda y Rojo. En este proyecto se usaron dos editores de texto: NetBeans, MySQL y VisualStudioCode. Se utilizó Java, SQL, html, css y JavaScript.
